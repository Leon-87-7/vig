Suggested design-system changes for Ownix

Main shift: move the system from “Video Intelligence Gateway / Operator’s Console” to “Ownix / Personal Index.” The current design is sharp and operational, but the copyright file asks for something quieter, more premium, and more personal.

1. Rename the System

  • From: vig — Video Intelligence Gateway
  • To: Ownix — Your internet, indexed
  • North Star: The Personal Index

2. Adjust the Product Metaphor

  • Replace “operator console” language with “private index,” “owned feed,” and “shared brain.”
  • Keep the product disciplined, but make it feel less industrial and less command-center-like.
  • Status should remain clear, but ownership state should become more prominent.

3. Revise the Color System

  • Keep the dark premium base, but soften the high-voltage orange.
  • Replace #f6921e with a quieter amber such as #d99a45.
  • Use amber for “save,” “add to Index,” “invite,” and “contribute to Brain.”

Creative North Star: "The Personal Index"

Ownix is not a command center. It is a calm, durable place for the internet a person wants to keep.
The interface should feel precise, private, and collected: dark premium surfaces, restrained amber
for acts of saving or contributing, and a softer Brain gradient used only where the shared knowledge
layer becomes visible.
