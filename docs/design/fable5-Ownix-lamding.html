Three layers. You decide where things live.

Every item in Ownix has a legible home. Ownership isn't a setting buried in a menu — it's the structure of the product.

Index

layer 1 / private

Everything you save lands here first. Videos are transcribed, articles summarized, repos mapped — searchable, permanent, and visible to exactly one person: you.

Private

Feed

layer 2 / owned

A return path through your own Index. No algorithm deciding what you see, no engagement loop — just the things you chose to keep, resurfaced when they're useful again.

Indexed

Brain

layer 3 / opt-in

The shared layer. Contribute an item's signal to the collective map, or don't. Contribution is always explicit, always per-item, and always reversible.

Shared to Brain

The Brain is what we know together.

When enough people contribute the same signal — a video worth the 40 minutes, a repo that actually holds up — the Brain surfaces it. It's a map shaped only by deliberate contribution, never by watch time.
