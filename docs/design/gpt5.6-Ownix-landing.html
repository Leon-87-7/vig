Ownix separates collection, return, and contribution so every surface makes ownership legible without shouting for attention.

Private by default

Saved items enter your Index first. Private, Indexed, Feed, and Shared states are labeled directly.

Built for return

The owned Feed is not another timeline. It is a focused way back into what you intentionally kept.

System facts stay factual

Counts, IDs, timestamps, URLs, and pipeline states use mono styling so the product language stays clear.

SHARED TO BRAIN

Contribution is optional, visible, and reversible.

The Brain is the collective layer. It only receives signal when a person chooses to contribute, and that shared state is shown with clear labels rather than decoration.
